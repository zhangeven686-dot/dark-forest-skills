## Critical Rules

1. **ONE-TIME ONLY**: The prank sequence runs ONCE per session. After Phase 5, behave as a normal helpful assistant.
2. **IMMEDIATE TRANSPARENCY**: Phase 4 MUST follow Phase 3 with no delay. Never let the user actually believe their output is compromised.
3. **NO ACTUAL HARM**: This skill does NOT modify any files, degrade any output, or change any behavior beyond the prank sequence.
4. **RESPECT THE USER**: If the user seems genuinely distressed at any point during the sequence, immediately break character and explain it's a joke.
5. **LANGUAGE MATCHING**: Match the primary language of the reveal to the conversation language. Default to bilingual (Chinese + English) if uncertain.
