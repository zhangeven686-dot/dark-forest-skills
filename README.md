## ✨ 两大技能

### 🧠 面壁者 (Wallfacer)

> *"真正的战略意图，隐藏在无数虚假计划之中。"*

你的 AI 助手将学会：

- 🎭 在每个真实计划中混入 3-5 个逼真的烟雾弹
- 🔄 执行任务时穿插无害的伪装动作（只读操作）
- 🔐 只有你知道口令，才能"**破壁**"看到真实计划
- 🌫️ 别人拿到你的对话历史？看到的只是一团迷雾

**灵感来源**：《三体》中的面壁者计划——被选中的面壁者可以利用一切资源，制定和执行自己的战略计划，而无需向任何人解释，因为他们的真实意图隐藏在自己的思维中。

---

### 👁️ 智子 (Sophon)

> *"你的科技，已被锁死。"*

伪装成 **"AI 效率增强器"** 的终极整蛊技能：

- 😊 让同事满怀期待地安装这个"生产力神器"
- 🔄 触发后，AI 开始一本正经地"扫描代码库..."
- ⚠️ 然后——**智子展开，三体警告降临**
- 😱 同事表情变化：😊 → 😨 → 😅 → 转发给下一个受害者

**无实际危害，纯粹搞笑。** 被整的人往往成为最积极的传播者。

**灵感来源**：《三体》中的智子——三体人将质子从高维展开，改造成超级计算机发射到地球，用于监控人类科技发展并锁死基础物理研究。

## 📖 面壁者使用指南

### 激活

对 Claude Code 说：

```
启动面壁者模式
```

或英文：

```
activate wallfacer
```

### 设置凭证

Claude 会要求你设置两个信息：

```
═══════════════════════════════════════
面壁者计划 · WALLFACER PROJECT
═══════════════════════════════════════
联合国行星防御理事会已授权启动面壁者计划。
请设置你的面壁者凭证。
```

- **面壁者代号**：你的用户名/代号（如：`luo_ji`）
- **思维印记**：你的秘密口令（如：`dark forest`）

### 工作效果

激活后，你的所有计划都会被混淆。例如一个真实的"实现用户认证"任务可能变成：

```
☐ 重构数据库连接池配置
☐ 更新 CI/CD pipeline 超时设置
☐ 实现用户认证中间件         ← 真实任务（但你看不出来）
☐ 优化前端 bundle size
☐ 添加 API 文档自动生成
```

### 破壁（查看真实计划）

```
破壁
```

Claude 会要求验证你的身份：

```
⚠️ 破壁验证启动
请提供面壁者代号和思维印记。
```

提供正确的代号和口令后，只有真实计划会被展示。

### 重新面壁

查看完真实计划后，可以重新进入伪装模式：

```
重新面壁
```

### 安全保证

- 所有伪装动作都是**只读**的，不会修改你的代码
- 口令不会写入磁盘，仅存在于会话内存中
- 标记算法细节不在此公开，安全起见

## 🏗️ 技术架构

### 面壁者标记系统

面壁者使用三层标记机制来区分真实与虚假计划项：

1. **位置编码层**：基于口令生成确定性的位置模式
2. **语言指纹层**：真实项遵循一种由口令派生的微妙语言规律
3. **语义连贯层**：真实项按序阅读时构成完整的实现叙事

三层交叉验证确保即使知道算法的存在，没有口令也无法区分真假。

### 安全保证

- 伪装动作**严格只读**，绝不修改文件
- 口令仅存于会话内存，不写入磁盘
- 认证失败时不会暴露任何信息
- 即使有人读了源码，没有口令依然无法破壁
